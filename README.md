# Under Pressure
Keep track of your blood pressure.

![alt tag](preview.png)

## Requirements
Webserver with PHP 5+

## Installation

### Visualize log data
- Clone repository to webserver
- Open web app in browser (e.g. http://example.com/under-pressure)
- To remove the example entries delete or empty file "log.csv"
